# Básico

## Começando 🎉

```bash
git init
```

## Verificando seu projeto 🆗

```bash
git status
```

## Adicionando mudanças ➕

```bash
git add
```

## Salvando mudanças ✅

```bash
git commit
```

## Verificando o histórico ⬅️

```bash
git log
```

# Git Flow 🌊

## Pegando mudanças mais recentes 🆕

```bash
git pull
```

## Criando uma branch 🌿

```bash
git branch nome_da_brach
```

## Mudando de branch ➡️

```bash
git switch nome_da_branch
```

## Criando e mudando de branch 💎

```bash
git switch -c nome_da_branch
```

## Subindo suas mundanças 🆙

```bash
git push
```

## Mesclando branches 🔀

```bash
git switch destino
git merge fonte
```
